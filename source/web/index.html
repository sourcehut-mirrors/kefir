Welcome to Kefir C17 compiler playground!

Please note the following:

  • The Kefir compiler is running entirely in the browser. In order to implement that, this page uses JavaScript, WebAssembly and browser worker threads. Upon the first compilation the WebAssembly binary is fetched and a worker thread is spawned. No local storage or cookies are used.
  • Musl libc headers are the only include files available in the compilation environment. Musl header files are stored and served by the backend.
